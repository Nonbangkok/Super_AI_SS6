# Super AI Engineer Season 6

This repository contains projects for the Super AI Engineer Season 6 training program. It focuses on a set of Mini-Hackathons for practicing Data Science, Computer Vision, NLP, and basic AI pipeline workflows.

## Project structure

- `Level-1/`
  - Main folder for the first-level Mini-Hackathons
  - Each challenge is stored in its own subfolder with notebooks and sample data

### Mini-Hackathon list

1. `Mini-Hackathon-1-Form-Data-to-Insight/`
   - A project for data analysis and converting form data into insights
   - Includes notebook: `Form-Data-to-Insight.ipynb`
   - Includes sample data: `Electricity-Consumption-by-District.csv`

2. `Mini-Hackathon-2-Thai-Election-OCR/`
   - A project for OCR and image processing to extract election-related information
   - Includes notebook: `Thai-Election-OCR-Pipeline.ipynb`

3. `Mini-Hackathon-3-FahMai-RAG-Challenge/`
   - A project for Retrieval-Augmented Generation (RAG)
   - Includes notebooks: `FahMai-RAG-Challenge-1.ipynb`, `FahMai-RAG-Challenge-2.ipynb`
   - Includes `data/` folder for questions, sample submission, and knowledge base files
     - `data/questions.csv`
     - `data/sample_submission.csv`
     - `data/knowledge_base/` contains policy and product information files

4. `Mini-Hackathon-5-Domain/`
   - A project for domain-specific model experiments and applications
   - Includes several notebooks such as:
     - `1-House-Recognition.ipynb`
     - `2-Word-Segmentation.ipynb`
     - `3-Sleep-Stage-Classification.ipynb`
     - `4-Heart-Disease-Prediction.ipynb`
     - `5-Thai-Image-Captioning.ipynb`
